Bright Eyed – Android eCommerce Application
Project Description

Bright Eyed is a modern Android-based eCommerce mobile application developed to provide users with a smooth and secure online shopping experience.
The application allows users to browse products, view detailed product information, add items to cart, place orders, and manage their shopping activity efficiently.

The app is integrated with Firebase for backend services such as authentication, real-time database/storage, and cloud-based data handling, making it scalable, secure, and easy to maintain.

This project demonstrates practical implementation of mobile application development, Firebase integration, UI/UX design, and real-world eCommerce functionalities.


Features:
User Registration and Login
Firebase Authentication
Product Listing and Categories
Product Details Page
Add to Cart Functionality
Order Placement System
Firebase Realtime Database / Firestore Integration
Product Images Storage using Firebase Storage
Search and Filter Products
User Profile Management
Clean and User-Friendly UI
Secure Cloud-Based Data Management
